# YouTube AI Factory (Upgraded Free System)

This repository contains the upgraded, production-ready n8n workflow for an automated YouTube content creation pipeline.

## 🚀 BEFORE YOU IMPORT (IMPORTANT)

Inside n8n you must:
* Install locally:
  * `ffmpeg`
  * `gtts-cli`
* Add credentials:
  * YouTube Data API (OAuth)
  * (Optional) OpenAI API

## 🔁 WHAT THIS VERSION DOES (UPGRADED)

Every 2 hours:
👉 Generates **3 videos automatically**
👉 Creates script + metadata
👉 Converts to voice
👉 Builds video using FFmpeg
👉 Uploads to YouTube
